# LUXE Auto Gallery

A world-class luxury car dealership platform built for the most discerning automotive collectors and enthusiasts.

## ✨ Features

- **Cinematic Experience:** Immersive hero sections with high-production automotive visuals and smooth Framer Motion transitions.
- **Advanced Inventory:** High-performance filtering and search system with grid/list view support.
- **Prestige Design:** A bespoke luxury dark theme featuring matte black, metallic gold, and silver chrome accents.
- **Componentized Architecture:** Fully modular React components built with Next.js 15+ and Tailwind CSS 4.
- **Admin Portal:** Comprehensive executive dashboard for inventory management, lead tracking, and performance analytics.
- **Full-Suite Pages:** Includes Home, Inventory, Car Details, About, Services, Blog, Testimonials, Contact, FAQ, and Auth.
- **Premium UI/UX:** Glassmorphism, premium typography (Playfair Display & Inter), and interactive hover effects.
- **Performance Optimized:** SEO-friendly, accessible, and high-speed frontend architecture.

## 🚀 Tech Stack

- **Frontend:** React, Next.js 15+, Tailwind CSS 4, Framer Motion
- **Icons:** Lucide React
- **Typography:** Google Fonts (Playfair Display, Inter)
- **Analytics:** Chart.js
- **Animations:** Framed Motion

## 🛠️ Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to explore the showroom.

## 📁 Project Structure

- `src/app`: Next.js App Router pages and layouts.
- `src/components`: UI components organized by layout and feature.
- `src/data`: Mock data for vehicles, blog posts, and testimonials.
- `src/lib`: Utility functions and shared libraries.
- `src/hooks`: Custom React hooks for animations and state.
